What is NIMS?

Software Engineering, Team NIMS is a mathematical game of strategy in which two players take turns removing objects from distinct heaps. On each turn, a player must remove at least one object, and may remove any number of objects provided they all come from the same heap. The goal of the game is to be the player to remove the last object.

Business Objectives

The following is the list of business objectives:

    Objective 1: User will be presented with an animated splash screen upon application start up, with the following:

    • It must only be implemented when the application has a cold start.
    • It must end as soon as the application has loaded in-order to save the users time.

    Objective 2: User will be ppresented with a homescreen, with the following:

    • Play Button
    • Game Settings
    • System Settings
    • How to Play button

    Objective 3: "Play" button must take the user to the game and allow the user to play the game with the settings that were selected, and the game must have the following:

    • Sub - par graphics
    • Intelligent A.I. (computer)
    • Buttin for changing turns
    • Animated images to play the game

    Objective 4: "Game Settings" functionality must be so that the following is displayed:

    • Number of rows
    • Difficulty Level
    • Starting Turn
    • Opponent (vs A.I. or another player)

    Objective 5: "How to Play" functionality must be so that:

    • Teaches the user the basics of the game and gives elementary tips on how to beat the A.I.
